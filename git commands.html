initialize a git repository
git init
show the status of the git repo
git status
add a file to git
git add filename
add all files to git (be careful...)
git add .
commit your changes
git commit -m"some message that makes sense"
upload your changes
git push origin master
download the changes
git pull origin master


create a new branch
git branch branchname
show available branches
git branch
switch to a branch
git checkout branchname
upload a branch
git push origin branchname
